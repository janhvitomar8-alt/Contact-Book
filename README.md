# Contact Book

A command-line Contact Book application built using Python.

## Features

* Add Contact
* View Contacts
* Search Contact
* Delete Contact
* Save Contacts to File
* Load Contacts from File
* Phone Number Validation
* Email Validation

## Concepts Used

* Python Functions
* Lists
* Dictionaries
* Loops
* Conditional Statements
* File Handling
* Exception Handling
* String Methods

## How to Run

1. Install Python.
2. Open a terminal in the project folder.
3. Run:

python contact_book.py
